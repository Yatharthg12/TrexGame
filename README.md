**Trex Game**

_Trex Game_ is a browser-based recreation of the classic T-Rex runner game popularized by Google Chrome when offline. The player controls a dinosaur and tries to survive as long as possible by avoiding obstacles.

The game is built using HTML, CSS, and JavaScript.

---

**Gameplay:**

1. Press the **spacebar** to jump  
2. Avoid cacti and other obstacles  
3. The game speed increases over time to make it more challenging  
4. The goal is to get the highest score possible

---

**Technologies Used:**

1. JavaScript  
2. HTML  
3. CSS  

---

**How to Run:**

1. Clone the repository:
git clone https://github.com/Yatharthg12/TrexGame.git
2. Open `index.html` in a modern web browser.

---

**Features:**

1. Classic T-Rex runner gameplay  
2. Responsive controls  
3. Increasing difficulty over time  
4. Score tracking
